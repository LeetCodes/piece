SAUCE 1 "AUGUST 2014" Linux "User Manuals"

NAME

sauce - displays Standard Architecture for Universal Comment Extensions meta data

SYNOPSIS

sauce [-V]

sauce files ...

DESCRIPTION

sauce parses the embedded SAUCE meta data packets in files, showing all the details for the detected data type and file type.

OPTIONS

-V --version
Display version information
